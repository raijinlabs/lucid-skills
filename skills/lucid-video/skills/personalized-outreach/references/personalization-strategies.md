# Personalization Strategies

## Tier 1: Basic (automated, no research)
- First name in greeting
- Company name in context scene
- Industry-matched social proof

## Tier 2: Enriched (with data lookup)
- Recent company news or funding round
- Specific role-based pain point
- Competitor comparison relevant to their stack

## Tier 3: Deep (with human input)
- Reference to specific LinkedIn post or tweet
- Mention mutual connection
- Custom solution demo for their use case

## Scaling Personalization
- Tier 1: 100+ videos/day (fully automated via lucid-prospect data)
- Tier 2: 20-50 videos/day (enrichment API + templates)
- Tier 3: 5-10 videos/day (human review + custom elements)

## Metrics to Track
- Open rate by personalization tier
- Response rate by CTA type
- Meeting booked rate by industry vertical
- A/B test: video length (15s vs 30s)
